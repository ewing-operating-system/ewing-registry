## Why GREEN Was Pulled Out
NOT scope creep. Resource conflict:
- MB-27 running Salesfinity VoIP (browser, mic, speaker, full CPU for parallel dialing)
- Claude Code couldn't run simultaneously without degrading call quality
- Customers couldn't hear Ewing
- GREEN came out as Claude machine but has broken screen + overheating
- Mac mini purchased as dedicated Claude/ClawdBot machine

## Why Mac Mini Got Scope Creep
- It was the ONLY available machine Friday-Sunday
- MB-27 was still set up for Monday's calls
- GREEN was shut down (overheating, broken screen)
- Everything got built on Mac mini because nothing else was available

## Revised Diagnosis
The pipeline works at both ends — calls are happening AND infrastructure is built — but the middle is missing. Data flows in but doesn't flow through.

## The Filter Going Forward
Does this help Ewing get a signed representation agreement faster? If the answer isn't immediate and obvious, it doesn't get built yet.
